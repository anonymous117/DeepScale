modules that need to be loaded:

* CUDA
* GCC
* Nsight-Systems

for running python, tensorflow ml code also need to load the modules:

* TensorFlow

to run with nvtx profiler first install the python package:

pip install --user nvtx

then use the annotations as explained here:

https://developer.nvidia.com/blog/nvidia-tools-extension-api-nvtx-annotation-tool-for-profiling-code-in-python-and-c-c/

profile code with srun:

srun -n1 -N1 nsys profile -f true -o my_report ./executable

to open the report:

nsys-ui

nsys stats my_report.qdrep
